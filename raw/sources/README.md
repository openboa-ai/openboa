# Raw Sources

`raw/sources/` stores imported source materials and provenance artifacts.

Suggested structure:

```text
raw/sources/
  inbox/
  library/
  assets/
```

- `inbox/`
  - newly collected source material that has not been normalized yet
- `library/`
  - curated immutable source library
- `assets/`
  - attached images and downloaded companion files

Do not rewrite source content here.
Use `wiki/` for synthesis and `docs/` for stable explanation.
