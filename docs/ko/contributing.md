## 문서 규칙

- 공식 문서는 `docs/` 하위에 작성
- 파일명은 lowercase kebab-case
- 스레드/PR 맥락은 공식 문서 본문에 넣지 않음
- 기술 근거 링크는 기술 문서에 배치
