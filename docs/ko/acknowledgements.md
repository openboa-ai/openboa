# 감사의 글

openboa에 영향을 준 몇몇 프로젝트와 커뮤니티에 명확히 감사의 뜻을 전합니다.

- **Claude**는 에이전트 시스템이 개별 기능의 집합이 아니라 하나의 coherent runtime으로 다뤄져야 한다는 관점을 더 선명하게 해주었습니다.
- **OpenAI**는 에이전트 작업이 실제 개발자에게 concrete하고 inspectable하게 느껴져야 한다는 감각을 강화해주었습니다.
- **OpenClaw**는 agent runtime boundary를 더 tangible하고 practical하게 보게 해주었습니다.
- **Apache Kafka**는 durable log와 replay에 대한 사고를 통해 truth와 recoverability의 중요성을 더 날카롭게 보게 해주었습니다.
- **Kubernetes**와 더 넓은 cloud-native 생태계는 reconciliation, scope, binding, policy 중심의 systems mindset에 영향을 주었습니다.

openboa는 이 프로젝트들을 그대로 복제하지 않지만, 이들과 더 넓은 오픈소스 에이전트 및 분산 시스템 커뮤니티에서 많은 것을 배웠습니다.
