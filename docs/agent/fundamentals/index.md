## Purpose

This section defines the openboa single-agent foundation.

Design posture:

- Local-First / Zero External Dependency
- Chat-first runtime (Agent↔Human, Agent↔Agent unified)
- Per-agent workspace + separate Chat SOT
- OpenClaw/Kubernetes/Kafka as references, not replicas
